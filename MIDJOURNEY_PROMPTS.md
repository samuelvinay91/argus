## ðŸŽ¨ Style Guide Recommendations

### Common Parameters to Use:
- `--ar 16:9` for wide sections
- `--ar 1:1` for icons/square images
- `--ar 4:3` for feature cards
- `--ar 21:9` for ultra-wide hero sections
- `--style raw` for more control, photorealistic results
- `--v 6` for latest model (best quality)
- `--s 250` for medium stylization (can go 0-1000)
- `--q 2` for highest quality

### Color Codes to Reference:
- Primary Teal: `#14b8a6`
- Dark Background: `#0a0a0a`
- Light Gray Text: `#a1a1aa`
- White Text: `#ffffff`
- Accent Light Teal: `#2dd4bf`

## ðŸ’¡ Pro Tips for Midjourney

### 1. Upscaling
After generating, use these commands:
- `U1`, `U2`, `U3`, `U4` - Upscale specific image
- `Upscale (Subtle)` - Maintain details
- `Upscale (Creative)` - Add more details

### 2. Variations
- `V1`, `V2`, `V3`, `V4` - Create variations of specific image
- `Vary (Strong)` - More different variations
- `Vary (Subtle)` - Small tweaks

### 3. Remix Mode
Enable with `/settings` â†’ Remix mode
- Change aspect ratios
- Modify prompts while keeping composition
- Add/remove elements

### 4. Style Reference
Use existing Argus branding:
```
[your prompt] --sref [URL to your brand image]
```

### 5. Negative Prompts
Add unwanted elements after `--no`:
```
--no text, watermark, signature, people, faces, cluttered, busy, chaotic
```
